## 코드 구상
* input 태그로 입력받은 문자열을 세 부분 (영단어, 숫자, 방향 순) 으로 나눠서 저장한다.
* 영단어는 한 글자씩 쪼개서 배열에 저장한다.
* 숫자는 입력받으면 문자열로 인식되므로, 데이터 타입을 숫자로 변환한다.
* 방향은 L,l,R,r 중으로 하나를 입력받는데, 입력받은 숫자가 음수면 방향을 반대로 바꾼다.
